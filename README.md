## Featured Projects

### Royal Chat — Production Persian RAG Assistant
*Production system @ Shaygansoft.* A retrieval-augmented generation assistant for clinic staff, deployed on a single Linux VM with fully local embedding inference (no external API). Hybrid retrieval (FAISS + BM25 via Reciprocal Rank Fusion + cross-encoder reranking), token-budgeted context packing, and real-time LLM token streaming.

**Highlights:** hit@5 **95%** · MRR **0.87** (126-question golden eval) · Dockerized · GitHub Actions CI/CD
**Tech:** Python • FastAPI • Hugging Face • FAISS • BM25 • DeepSeek API • Docker

---

### Royal Bina — Clinic Business Intelligence Pipeline
*Production system @ Shaygansoft.* End-to-end on-premises BI pipeline transforming clinic CRM and ERP databases into a SQL Server analytics layer, with Persian Metabase dashboards and automated nightly refresh.

**Highlights:** **~266K** service-line records modeled · ~34 dashboard cards · caught a NULL-handling defect understating revenue **~48%**
**Tech:** SQL Server • T-SQL • ETL • Metabase • PowerShell • Dimensional Modeling

---

### NLP Pipeline for Amazon Reviews
Transformer-based multi-class sentiment system trained on 800K+ Amazon reviews; fine-tuned DeBERTa, RoBERTa, and DistilBERT, with two semantic-search approaches.

**Highlights:** **78.5% F1** (DeBERTa)
**Tech:** PyTorch • Hugging Face • Sentence Transformers • FastText • BERT family

---

### Real Estate Price Prediction
Large-scale ML pipeline for price prediction, statistical analysis, and market segmentation on 1M+ real estate records.

**Highlights:** **R² = 0.75** · hypothesis testing (t-test, Mann–Whitney U) · K-Means / DBSCAN · Bayesian tuning
**Tech:** LightGBM • Random Forest • scikit-learn • SciPy • Pandas
